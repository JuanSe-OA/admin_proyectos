Admin Proyectos - Project Management Suite
Admin Proyectos es una herramienta de gestión diseñada para equipos que buscan optimizar su productividad. La plataforma facilita la planificación, ejecución y monitoreo de proyectos, asegurando que los objetivos se cumplan dentro de los plazos establecidos.

Características Principales
Panel de Control (Dashboard): Visualización centralizada de métricas clave, estado de los proyectos y tareas pendientes.

Gestión de Equipos: Sistema de roles para asignar líderes de proyecto y colaboradores, facilitando la delegación de responsabilidades.

Seguimiento de Tareas: Creación y gestión de tableros de actividades con estados personalizados (Pendiente, En Proceso, Finalizado).

Control de Plazos: Gestión de fechas de entrega y alertas para hitos críticos del proyecto.

Reportes de Progreso: Generación de informes sobre el rendimiento del equipo y el cumplimiento de metas.

Stack Tecnológico
Frontend:  React con estilos modernos y responsive.

Backend:  Python para la gestión de la lógica de negocio.

Framework: FastApi

Base de Datos:  MySQL  para el almacenamiento persistente de la estructura de proyectos.

Autenticación: Gestión de sesiones segura para el acceso administrativo.

Instalación
Clonar el repositorio: git clone https://github.com/JuanSe-OA/admin_proyectos.git

Instalar las dependencias necesarias: npm install (o el comando correspondiente a tu lenguaje).

Configurar el archivo de entorno .env con las credenciales de la base de datos.

Ejecutar las migraciones o scripts de inicio.

Iniciar la aplicación: npm start.
